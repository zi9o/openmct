Methods

destroy()

Release any resources associated with this view.

View implementations should use this method to detach any listeners or release other resources that are no longer necessary once a view is no longer used.

Populate the supplied DOM element with the contents of this view.

View implementations should use this method to attach any listeners or acquire other resources that are necessary to keep the contents of this view up-to-date.
